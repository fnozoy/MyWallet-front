# MyWallet-front
front in React JS

### `Installation tips`<br />
install nodejs <br />
> npm install -g yarn <br />
> npm install -g create-react-app <br />
> create-react-app mywallet <br />
> yarn start <br />
> yarn add bootswatch <br />
> yarn add react-router-dom <br />
> yarn add axios <br />
> yarn add toastr <br />
> yarn add currency-formatter <br />
